# Data Dictionary

## Performance Dataset

| Variable | Description |
|---|---|
| player | Player name |
| squad | MLS club/team |
| position | Player position |
| age | Player age |
| matches | Matches played |
| starts | Number of starts |
| minutes | Minutes played |
| goals | Goals scored |
| assists | Assists recorded |

## Salary Dataset

| Variable | Description |
|---|---|
| first_name | Player first name |
| last_name | Player last name |
| club | MLS club |
| position | Player position |
| base_salary | Annual base salary |
| guaranteed_compensation | Guaranteed annual compensation |
